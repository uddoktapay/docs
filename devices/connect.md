# Connect Device

To connect a new Android device:

1. Install the **Payment Automation App** (only for Bangladeshi users).  
2. Open the app and grant all requested SMS permissions.  
3. In the dashboard, go to **Devices → Connect Android App**.  
4. Scan the QR code shown on screen.  

Once connected, the device will appear in the **Device List** with details such as:

- Device ID & Name  
- Model  
- Android Version  
- App Version  
- Status (Connected)
