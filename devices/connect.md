# Connect Device

To connect a new Android device:

1. Install the **Payment Automation App**
2. Insert the **bKash or Nagad SIM** you will use for receiving payments into this phone.  
   - The same SIM must be in the same phone that runs the app, because SMS is read directly from the device.  
3. Open the app and grant all requested SMS permissions.  
4. In the dashboard, go to **Devices → Connect Android App**.  
5. Scan the QR code shown on screen.  

Once connected, the device will appear in the **Device List** with details such as:

- Device ID & Name  
- Model  
- Android Version  
- App Version  
- Status (Connected)  

::: warning
For auto verification to work, the app must always be running in the background on the phone.  
Make sure battery saver is disabled for the app.
:::
