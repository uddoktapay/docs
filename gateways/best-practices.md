# Gateway Best Practices

- **Enable only what you use** — keep unused gateways disabled to avoid confusion.  
- **Sandbox first** — test transactions before enabling live mode.  
- **Name clearly** — use display names customers recognize (e.g., “bKash Personal” instead of “Bkash_01”).  
- **Set proper limits** — use minimum/maximum amounts to control transaction sizes.  
- **Verify fees** — ensure fixed and percentage charges match your provider agreement.  
- **Bank gateways** — always request proof fields like Trx ID for manual payments.  
- **Monitor status** — toggle off gateways temporarily during provider downtime.  

::: tip
Revisit your gateway list monthly to confirm charges, credentials, and active status are correct.
:::
