# Create Bank Gateway

Bank Gateways are used for offline transfers. They collect user proof and are approved from the Payments module.

Steps:

1. Click **New bank gateway**.
2. Fill **Gateway Information**:
   - **Gateway Name** and **Display Name**
   - **Currency**
   - **Minimum / Maximum Amount**
   - **Fixed / Percentage Charges**
3. Upload a **Logo** and optional **Description**.
4. Fill **Bank Information** (e.g., Bank Name, Account Holder, Account Number, Branch, Routing, SWIFT/BIC).
5. Configure **User Input Fields** for checkout (e.g., Transaction ID, payment slip upload).
6. Save.

::: tip
Bank gateways can be **duplicated** if you need similar variations. Use **Replicate** on the list page.
:::
