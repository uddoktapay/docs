# Edit & Configure Gateways

Open **Edit** to change amounts, fees, names, currency, and provider configuration.

For automatic gateways, you may see:

- **Sandbox** enable/disable
- **Credentials** (for example, Username, Password, App Key, App Secret)
- Other provider-specific parameters

::: tip
Changes take effect immediately. Ensure credentials are valid before enabling the gateway.
:::
