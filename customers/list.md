# Customer List

The list shows all customers with name, email, and phone number.

## Columns

You can choose which columns are visible from the column selector. Typical fields:

- Name
- Email
- Phone
- Address
- City
- State
- Postcode
- Country

The table supports sorting on common fields such as **Name**, **Email**, and **Phone**.

::: tip
Use the column selector to hide fields you do not need. This makes the list easier to scan.
:::

## Search

Use the search bar to find customers by name, email, or phone.

## Actions

Each row provides **Edit** and **Delete**.

::: warning
Deleting a customer does not delete related payments or invoices. Those records remain, but the customer link is removed.
:::
