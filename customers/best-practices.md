# Customer Best Practices

- **Use email as the primary identifier**  
  Ensure each customer has a valid email. This helps avoid duplicate records.

- **Keep phone numbers consistent**  
  Use international format (e.g., `+1 555 123 4567`) to reduce duplication.

- **Leverage imports carefully**  
  Always back up existing data before running a bulk import.

- **Do not delete unnecessarily**  
  Instead of deleting, update records to preserve history and reporting.

- **Segment customers by brand**  
  In a multi-brand setup, ensure customers are associated with the correct brand context.
