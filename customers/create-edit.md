# Create & Edit Customers

Use **New customer** to add a record manually. The required fields are:

- **Name**
- **Email**
- **Phone**

Optional fields include:

- Address
- City
- State
- Postcode
- Country

You can update customer details at any time using **Edit**.

::: tip
Keep customer information accurate and complete. This improves reporting and helps prevent duplicates.
:::
