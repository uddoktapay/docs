# Date Ranges

Reports can be filtered by time period using the buttons above the report:

- **Daily**
- **Weekly**
- **Monthly**
- **Yearly**
- **Custom Range**

## Custom Range

For a custom range:

1. Click **Custom Range**.
2. Select a **Start Date** and **End Date**.
3. Click **Apply** to update the report.

::: tip
Use custom ranges for financial reconciliations or campaign-specific reporting.
:::
