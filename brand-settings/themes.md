## Introduction

Themes control the appearance of your checkout pages, payment links, and gateway selection interfaces. Customize colors, buttons, navigation, and add custom CSS/JavaScript to match your brand identity.

Access Themes from **Brand Settings → Themes** to manage and customize your checkout appearance.

## Theme List

The themes list displays all available themes with preview images.

## Customize Theme

Click **Customize** to modify theme appearance and behavior.

## Custom Theme Development

Need a completely custom theme or want to modify theme structure beyond color and code customization?

::: tip
**For Developers:**  
Create custom themes with full control over layout, components, and functionality. Custom themes integrate seamlessly and appear in the themes list alongside built-in options.

See the [Developer Guide](/developers/custom-themes) for implementation details, including:
- Creating custom theme packages
- Defining theme configuration options
- Building theme layouts and components
- Registering custom themes
- Theme structure and file organization
- Adding custom configuration sections


**Common use cases:**
- Industry-specific checkout designs
- Multi-language theme variations
- Accessibility-focused themes
- Mobile-first custom layouts
- White-label theme packages
:::
