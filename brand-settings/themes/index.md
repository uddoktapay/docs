# Themes

The **Themes** section allows you to manage and customize the look of your checkout, gateway, and payment link pages.

## Features
- Browse available themes.
- Activate/deactivate themes per brand.
- Customize visual elements (colors, buttons, navigation, etc.).
- Add custom CSS or JavaScript for advanced control.

## Managing Themes
- Go to **Brand Settings → Themes**.
- Available themes will be shown as cards.
- Each theme provides options:
  - **Customize** → Open customization settings.
  - **Deactivate** → Disable the theme.

> 💡 Example: The **Default Theme** is a clean and optimized checkout theme with customizable options.
