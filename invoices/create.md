# Create Invoice

To create a new invoice:

1. Click **New invoice**.
2. Choose a **Customer** (create one if needed).
3. Set the **Currency**, **Due Date**, and initial **Status** (usually *Unpaid*).
4. Add **Items**:
   - Description
   - Quantity
   - Amount
   - Discount (optional)
   - VAT (optional)
5. Review **Totals** (shipping, VAT, discounts).
6. Add **Notes** if necessary.
7. Save the invoice.

After saving, the invoice appears in the list and can be viewed, edited, or sent to the customer.
