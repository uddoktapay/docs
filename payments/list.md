# Payments List

The Payments list page shows all transactions in a tabular view. You can browse, search, filter, and adjust visible columns.  

## Columns

You can customize which columns are visible in the table. Click the column selector icon and choose fields such as:

- Id  
- Customer  
- Gateway  
- Amount / Net Amount  
- Refunded Amount  
- Transaction Id  
- Phone Number  
- Date  
- Status  

::: tip
Use **Apply columns** to save your preferences or **Reset** to restore the default set.
:::

## Filters

You can refine the payments list using filters:

- **Status** – All, Completed, Pending, Refunded, Partially Refunded, Failed, Voided.  
- **Gateway** – Filter by any configured gateway (e.g., bKash Merchant).  
- **Created From / Created Until** – Limit results by a date range.  

::: tip
Filters can be combined. For example, filter by **Completed** and **bKash Merchant** to audit a single gateway’s successful payments within a time period.
:::

## Search

A quick search bar is available for finding payments by **Customer**, **Gateway**, or **Transaction ID**.
