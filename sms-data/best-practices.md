# SMS Data Best Practices

- **Keep the Paymently App connected** — reduces the need for manual entry.  
- **Approve quickly** — so user payments can be verified without delay.  
- **Review suspicious entries** — check for typos, wrong amounts, or fake IDs.  
- **Clear old data** — trash or archive used records to keep the list clean.  
- **Secure devices** — make sure the phone running the Paymently App is safe and monitored.  

::: tip
If your gateways rely on SMS verification, SMS Data is critical. Regularly review and approve to keep payment flows smooth.
:::
