# Manual Entry

Use Manual Entry when:

- The phone running Paymently was offline.  
- The SMS was not forwarded.  
- Automatic parsing failed.  

Instead of pasting the full SMS, enter the transaction details manually:

- **Payment Method**  
- **Type** (Agent, Personal, Merchant)  
- **Amount**  
- **Phone Number**  
- **Transaction ID**  
- Optional: Reference, Note  
- **Status** (Approved or Awaiting Review)  

::: tip
Use Manual Entry as a fallback. If the SMS text is available, always try **Automatic Entry** first.
:::
