# Getting Started with UddoktaPay

Welcome to UddoktaPay! 
This guide will help you set up UddoktaPay for the first time and understand the basics of how it works.

---

## 1. Choose Your Setup

- **Hosted Version**  
  No installation required — sign up and start using instantly.  
  👉 [Get UddoktaPay Hosted](https://my.uddoktapay.com/store/uddoktapay-hosted)

- **Self-Hosted Version**  
  Install UddoktaPay on your own VPS or server for complete control.  
  👉 [Installation Guide](/guide/installation)

---

## 2. Check System Requirements

Before installing, make sure your server meets all prerequisites.  
👉 [View System Requirements](/guide/system-requirements)

---

## 3. Quick Installation (Self-Hosted)

1. **Download the installer file** from UddoktaPay service details.
2. **Upload files** to your server.  
3. **Run the installation wizard** to set up database and admin account.  
4. **Set up Cron Jobs** for scheduled tasks.  
5. **Run Queue Workers** for payment processing & notifications.  

👉 Full details: [Installation Guide](/guide/installation)

---

## 4. First Steps After Install

- Update your **Brand Settings** (logo, colors, mail settings, etc.).  
- Configure **Gateways** (bKash, Nagad, Rocket, cards, or custom).  
- Set up **Notification Channels** (Email, Slack, Discord, etc.).  
- Create your first **Invoice** or **Payment Link**.
