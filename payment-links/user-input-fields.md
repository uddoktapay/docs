# User Input Fields

Add fields to collect extra data at checkout.

For each field:

- **Form Type** — input type (Text, Email, Number, etc.)
- **Field Name** — label shown to the customer
- **Is Required** — enforce completion before payment

Reorder fields using the move handles.

::: tip
Keep required fields minimal to reduce friction and improve conversion.
:::
