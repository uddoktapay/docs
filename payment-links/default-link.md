# Default Link

The **Default Link** opens a generic checkout form for your brand. Use it when you want a single link that always accepts payments and can be shared anywhere.

**Form fields (default):**

- Full Name  
- Email  
- Mobile Number  
- Amount  
- Reference (optional)  

The default form is designed for quick payments and donations. It does not use product-level settings like quantity or per-item description.

::: warning
The Default Link does not include product controls (fixed price, inventory, expiry per product).  
Create a **custom Payment Link** when you need those features.
:::
