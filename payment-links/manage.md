# Manage Links

In **Payment Links** you can:

- **Search** by product name/description
- **Sort** by name, amount, quantity, status
- **Toggle** status inline
- **Edit / Copy / Delete** a link
- **Paginate** and adjust items per page
- **Bulk-select** rows for batch actions (when enabled)

::: tip
Adopt a naming convention (e.g., `Campaign – Product – Variant`) to keep lists clean at scale.
:::
