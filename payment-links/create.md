# Create a Payment Link

Go to **Payment Links → New payment link** and complete the form.

## Required fields

- **Product name**
- **Quantity**
- **Currency**
- **Amount**

## Optional fields

- **Product description**
- **Expire Date** (`dd/mm/yyyy`)
- **Status** (toggle)

### User input fields

Add custom fields your customers must fill in during checkout (see the dedicated page for details).

::: tip
Use clear names (e.g., “Workshop Seat – Early Bird”). It makes searching and reporting easier.
:::

### Save

- **Create** — save and return to list  
- **Create & create another** — save and open a new blank form
