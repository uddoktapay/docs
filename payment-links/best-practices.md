# Usage Tips

## Which link should I use?

- **Default Link**  
  Use when you want a single, evergreen link for quick payments or donations.  
  Flexible amount entry, but no product-specific settings.

- **Custom Payment Link**  
  Use when you need fixed pricing, product description, quantity control, expiry date, or custom input fields.  
  Best for selling specific products or campaigns.

::: tip
A good rule of thumb: **Default for donations, Custom for products.**
:::

---

## General guidance

- Keep the form simple; fewer required fields increase conversion.  
- Use **Expire Date** for limited-time offers.  
- Name links consistently for better search and reporting.  
- Test a small payment before sharing publicly.  
- Prefer the **Default Link** when you need a single evergreen URL.  
- Keep inventory accurate if quantity matters.
